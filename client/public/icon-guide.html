아이콘 및 OG 이미지 가이드

운명의 매트릭스 검사 웹사이트에 사용되는 모든 아이콘 및 OG 이미지 파일에 대한 가이드입니다.

파일 구조

SVG 원본 파일:

  • /public/icon-simple.svg - 기본 파비콘 (타로 카드, 별, 달 아이콘)
  • /public/og-simple.svg - 심플한 OG 이미지 (텍스트 및 별 장식만)

PNG 변환 파일:

  • /public/icon-192.png - 192x192 PNG 아이콘 (웹 앱 아이콘)
  • /public/icon-512.png - 512x512 PNG 아이콘 (웹 앱 아이콘)

아이콘 업데이트 방법

SVG 원본 파일을 수정한 후 PNG 파일들을 업데이트하려면:

  1. SVG 파일 (icon-simple.svg)을 원하는 디자인으로 수정합니다.
  2. 루트 디렉토리에 있는 update-icons.bat 스크립트를 실행합니다: 
    .\update-icons.bat
  3. 스크립트가 자동으로 SVG를 PNG로 변환합니다.

참고: SVG를 PNG로 변환하기 위해서는 Inkscape가 설치되어 있으면 좋습니다. 설치되어 있지 않은 경우, 스크립트는 PowerShell로 기본적인 변환을 시도하지만 품질이 떨어질 수 있습니다.

HTML 메타태그

현재 다음 메타태그들이 index.html에 설정되어 있습니다:

<!-- Icons -->
<link rel="icon" href="/favicon.ico" />
<link rel="icon" type="image/svg+xml" href="/icon-simple.svg" />
<link rel="apple-touch-icon" sizes="180x180" href="/icon-simple.svg" />
<link rel="icon" type="image/png" sizes="192x192" href="/icon-192.png" />
<link rel="icon" type="image/png" sizes="512x512" href="/icon-512.png" />

<!-- Open Graph / Facebook -->
<meta property="og:image" content="https://destiny33.site/og-simple.svg" />

<!-- Twitter -->
<meta name="twitter:image" content="https://destiny33.site/og-simple.svg" />
